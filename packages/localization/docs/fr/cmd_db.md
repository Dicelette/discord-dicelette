# Commandes liées à la feuille de personnage

- </dbroll:{{- dbroll}}> : Lance un dé de statistiques, à partir d'une feuille de personnage.
- </dbd:{{- dbd}}> : Lance un dé enregistré dans votre feuille de personnage.
- </graph:{{- graph}}> : Permet de générer un graphique à partir des statistiques d'un personnage.
- </display:{{- display}}> : Permet de consulter les statistiques et les dés d'un personnage.
- </calc:{{- calc}}> : Permet d'effectuer des calculs avec les statistiques d'un personnage.
